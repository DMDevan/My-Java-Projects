Java Beginner Projects
-----------------------
This repository contains several small Java programs I created during my early stages of learning the language. Each project has been revisited and improved to reflect my current understanding of clean code, readability, and foundational programming practices. The projects are organized into packages inside the "src" directory.

-----------------------

|BASIC GAMES
    - A set of simple, text‑based console games designed to practice user input, decision‑making, and basic program flow.

1.) RockPaperScissors
    - A classic game where the user plays against the computer. Demonstrates random number generation, conditional logic, and comparing user choices.

2.) HiLow 
    - A number‑guessing game where the user tries to identify a randomly chosen number between 1 and 100. Highlights loops, comparisons, and interactive prompts.

3.) MultipleLeapYear 
  A small utility that checks whether a given year is a leap year. Includes input validation and clear logical conditions.

-----------------------

|MUTATIONS
    - Program focused on simple string operations and introductory Java concepts.

1.) CharacterCount  
    - Displays a sentence and reports how many characters it contains. Demonstrates string handling and basic output formatting.

-----------------------

|TEMPERATURE
Program involving basic mathematical conversions.

1.) CelsiusConverter  
  Converts a temperature from Fahrenheit to Celsius. Shows arithmetic operations and user input handling.

-----------------------

|GAME
    - A more advanced project that introduces graphics, movement, and real‑time interaction.

1.) CoinGame 
    - A small 2D game where the player controls a blue square and collects yellow “coins” scattered around the screen. A progress meter fills as coins are collected, and a message appears once all coins have been gathered. This project demonstrates drawing shapes on a window, detecting collisions, updating the screen in real time, and managing simple game logic.
